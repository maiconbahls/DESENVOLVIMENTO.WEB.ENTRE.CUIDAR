# Guia de Uso - Plataforma EntreCuidar

Este guia fornece as instruções básicas para administradores e clientes utilizarem a plataforma EntreCuidar de forma eficiente.

---

## 1. Acesso Administrador (Gestão e Auditoria)
**Usuário:** `adm@entrecuidar.com`

O painel administrativo é o centro de inteligência onde você gerencia novos leads, realiza diagnósticos e emite propostas.

### Fluxo de Trabalho:
1.  **Login:** Acesse a página inicial e utilize as credenciais de administrador.
2.  **Gestão Operacional:** 
    *   Aqui você vê todas as empresas cadastradas que responderam ao formulário inicial.
    *   **Botão "Entrevista":** Clique para realizar o diagnóstico profundo com o cliente.
    *   **Botão "Orçamento":** Após o diagnóstico, utilize este botão para definir os valores e enviar a proposta comercial.
3.  **Relatórios de Entrevistas (B.I.):**
    *   Acesse esta aba para visualizar os relatórios detalhados gerados automaticamente pelo sistema.
    *   Utilize o botão **"Exportar PDF"** dentro de cada relatório para enviar o diagnóstico técnico ao cliente.
4.  **Base de Cadastros:** Lista completa de empresas para consulta rápida de contatos.

---

## 2. Acesso Cliente (Hub de Soluções)
**Usuário:** E-mail corporativo cadastrado (Ex: `empresa@exemplo.com.br`)

O Hub do Cliente é uma área exclusiva onde ele acompanha a evolução da parceria e aprova as propostas enviadas.

### Funcionalidades:
1.  **Painel de Controle:** Visualização do status atual da conta e indicadores de economia/gestão.
2.  **Proposta Comercial:**
    *   O cliente deve clicar em **"Ver Proposta"** para analisar os valores e serviços oferecidos.
    *   **Aprovação:** Caso esteja de acordo, o cliente clica em **"Aceitar Proposta"**, o que formaliza o início dos serviços.
3.  **Chat de Suporte:** Canal direto para tirar dúvidas com a equipe EntreCuidar em tempo real.
4.  **Relatórios de Diagnóstico:** O cliente tem acesso aos resultados simplificados do seu nível de risco operacional.

---

## Dicas de Visualização (Mobile e Desktop)
*   A plataforma é **Totalmente Responsiva**. No celular, as tabelas se transformam em cartões fáceis de ler.
*   Para salvar qualquer relatório como PDF no computador:
    1. Clique no botão **"Exportar PDF"** ou use `Ctrl + P`.
    2. No destino da impressora, escolha **"Salvar como PDF"**.
